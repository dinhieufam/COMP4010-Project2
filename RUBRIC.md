# Project 2 — Grading Rubric

**Data Stories: Building Interactive Dashboards with Python Shiny**

Total weight: **100%** — Proposal (25%) · Presentation, Demo & Final Project (75%).

Each criterion is scored on a 4-level scale. Convert the level to a fraction of the criterion's weight: **Excellent = 100%**, **Good = 80%**, **Satisfactory = 60%**, **Needs Improvement = ≤40%**.
